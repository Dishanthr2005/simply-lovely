# Simply Lovely - Max Verstappen Red Bull F1 Page

A modern React landing page showcasing **Max Verstappen** on the **Red Bull Racing** car with a clickable **“SIMPLY LOVELY”** title that plays a Max Verstappen edit video in fullscreen.

---

## Features

- Fullscreen **background image** of Max Verstappen + Red Bull Racing.
- **Centered title** with golden gradient text and subtle shadow.
- Clicking the title triggers:
  - **Browser fullscreen** (like pressing F11)
  - **YouTube video autoplay** with sound
- Clean, responsive, and modern design.
- Works on Chrome, Firefox, Safari, Edge (supports Fullscreen API).
